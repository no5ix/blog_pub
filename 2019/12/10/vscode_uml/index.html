еӨҮжіЁпјҡ

  1. Markdown All in One,VSCode дёӯж”ҜжҢҒ Markdown(й”®зӣҳеҝ«жҚ·й”®гҖҒзӣ®еҪ•гҖҒиҮӘеҠЁйў„и§Ҳзӯү)
  2. Markdown Preview Enhanced еҸҜд»ҘеҜ№ Markdown еҒҡеўһејәйў„и§Ҳ, жҜ”еҰӮж”ҜжҢҒеҗ„з§Қз»ҳеӣҫзӯү
  3. PlantUML, дёҖж¬ҫеҫҲејәеӨ§зҡ„пјҢе№¶дё”еҸҜд»Ҙз»ҳеҲ¶еҗ„з§ҚеӣҫеҪўзҡ„и„ҡжң¬иҜӯиЁҖгҖӮйңҖиҰҒе®үиЈ… java

йңҖиҰҒдёӢиҪҪзҡ„иҪҜд»¶

  1. java : еӣ дёәoracleдёӢиҪҪjdk-8йңҖиҰҒзҷ»йҷҶдәҶ, еҫҲйә»зғҰ, жүҖд»Ҙе»әи®®еңЁзҷҫеәҰзӣҙжҺҘжҗңзҙўjdk-8дёӢиҪҪз„¶еҗҺеҺ»зұ»дјјдәҺдёӯе…іжқ‘дёӢиҪҪз«ҷиҝҷз§ҚзҪ‘з«ҷеҺ»дёӢдёҖдёӘ, з„¶еҗҺжҢүз…§дёӢеӣҫй…ҚзҪ®зҺҜеўғеҸҳйҮҸ
  2. graphviz : еңЁ http://graphviz.org/дёӢиҪҪд№ӢеҗҺе®үиЈ…пјҢ з„¶еҗҺй…ҚзҪ®зҺҜеўғеҸҳйҮҸ
  3. plantuml.jar : еңЁ http://plantuml.com/ дёӢиҪҪ plantuml.jarпјҢ з„¶еҗҺй…ҚзҪ®зҺҜеўғеҸҳйҮҸ
  4. йҮҚеҗҜз”өи„‘

й…Қе®Ңд№ӢеҗҺзұ»дјјеҰӮдёӢеӣҫ, жіЁж„ҸеҺ»йҷӨзҺҜеўғеҸҳйҮҸи·Ҝеҫ„йҰ–е°ҫеӨҡдҪҷзҡ„з©әж ј, зү№еҲ«жҳҜpathзҡ„жңҖеүҚж–№, еҗҰеҲҷж— жі•жӯЈзЎ®иҜҶеҲ«.

еӨҮжіЁпјҡGraphviz жҳҜејҖжәҗеӣҫеҪўеҸҜи§ҶеҢ–иҪҜд»¶гҖӮеӣҫеҪўеҸҜи§ҶеҢ–жҳҜе°Ҷз»“жһ„дҝЎжҒҜиЎЁзӨәдёәжҠҪиұЎеӣҫеҪўе’ҢзҪ‘з»ңзҡ„еӣҫиЎЁзҡ„дёҖз§Қж–№ејҸгҖӮе®ғеңЁзҪ‘з»ңпјҢз”ҹзү©дҝЎжҒҜеӯҰпјҢиҪҜд»¶е·ҘзЁӢпјҢж•°жҚ®еә“е’ҢзҪ‘йЎөи®ҫи®ЎпјҢжңәеҷЁеӯҰд№ д»ҘеҸҠе…¶д»–жҠҖжңҜйўҶеҹҹзҡ„еҸҜи§ҶеҢ–з•ҢйқўдёӯжңүйҮҚиҰҒзҡ„еә”з”ЁгҖӮ
