и·Ёе№іеҸ°ејҖеҸ‘д№ӢCMakeз¬”и®°

Posted on 2018-04-21 | In Misc
CrossPlatform Compile Make

fec intro

дёҖдёӘиҪ»йҮҸзә§зҡ„жёёжҲҸжңҚеҠЎеҷЁеј•ж“Һ

еӣ дёәиҝ‘жқҘйңҖиҰҒжҠҠдёҖдәӣ Linux йЎ№зӣ®иҪ¬еҲ° Windows дёҠжқҘејҖеҸ‘, жүҖд»ҘжңүдәҶдёҖдәӣи·Ёе№іеҸ°ејҖеҸ‘зҡ„з¬”и®°, жӯӨзҜҮи®ІCMake, дҫӣд»ҘеҗҺжҹҘйҳ….

CMakeд»Ӣз»Қ

дҪ жҲ–и®ёеҗ¬иҝҮеҘҪеҮ з§Қ Make е·Ҙе…·пјҢдҫӢеҰӮ GNU Make пјҢQT зҡ„ qmake пјҢеҫ®иҪҜзҡ„ MS nmakeпјҢBSD MakeпјҲpmakeпјүпјҢMakeppпјҢзӯүзӯүгҖӮиҝҷдәӣ Make е·Ҙе…·йҒөеҫӘзқҖдёҚеҗҢзҡ„и§„иҢғе’Ңж ҮеҮҶпјҢжүҖжү§иЎҢзҡ„ Makefile ж јејҸд№ҹеҚғе·®дёҮеҲ«гҖӮиҝҷж ·е°ұеёҰжқҘдәҶдёҖдёӘдёҘеі»зҡ„й—®йўҳпјҡеҰӮжһңиҪҜд»¶жғіи·Ёе№іеҸ°пјҢеҝ…йЎ»иҰҒдҝқиҜҒиғҪеӨҹеңЁдёҚеҗҢе№іеҸ°зј–иҜ‘гҖӮиҖҢеҰӮжһңдҪҝз”ЁдёҠйқўзҡ„ Make е·Ҙе…·пјҢе°ұеҫ—дёәжҜҸдёҖз§Қж ҮеҮҶеҶҷдёҖж¬Ў Makefile пјҢиҝҷе°ҶжҳҜдёҖд»¶и®©дәәжҠ“зӢӮзҡ„е·ҘдҪңгҖӮ

е°ұжҳҜй’ҲеҜ№дёҠйқўй—®йўҳжүҖи®ҫи®Ўзҡ„е·Ҙе…·пјҡе®ғйҰ–е…Ҳе…Ғи®ёејҖеҸ‘иҖ…зј–еҶҷдёҖз§Қе№іеҸ°ж— е…ізҡ„ CMakeList.txt ж–Үд»¶жқҘе®ҡеҲ¶ж•ҙдёӘзј–иҜ‘жөҒзЁӢпјҢз„¶еҗҺеҶҚж №жҚ®зӣ®ж Үз”ЁжҲ·зҡ„е№іеҸ°иҝӣдёҖжӯҘз”ҹжҲҗжүҖйңҖзҡ„жң¬ең°еҢ– Makefile е’Ңе·ҘзЁӢж–Үд»¶пјҢеҰӮ Unix зҡ„ Makefile жҲ– Windows зҡ„ Visual Studio е·ҘзЁӢгҖӮд»ҺиҖҢеҒҡеҲ°вҖңWrite once, run everywhereвҖқгҖӮ

жҳҫз„¶пјҢ

. . .

CMake жҳҜдёҖдёӘжҜ”дёҠиҝ°еҮ з§Қ make жӣҙй«ҳзә§зҡ„зј–иҜ‘й…ҚзҪ®е·Ҙе…·гҖӮдёҖдәӣдҪҝз”Ё CMake дҪңдёәйЎ№зӣ®жһ¶жһ„зі»з»ҹзҡ„зҹҘеҗҚејҖжәҗйЎ№зӣ®жңү VTKгҖҒITKгҖҒKDEгҖҒOpenCVгҖҒOSG зӯүгҖӮ

CMakeдёҖдәӣжңүз”Ёзҡ„зҪ‘з«ҷ

  • CMakeе®ҳзҪ‘
  • CMakeе…Ҙй—Ёе®һжҲҳ
  • иҝӣйҳ¶

е°Ҷbuildе’ҢйЎ№зӣ®жәҗж–Үд»¶еҲҶзҰ»зҡ„ж–№жі•

еҒҮи®ҫйЎ№зӣ®Aзҡ„ж №зӣ®еҪ•дёӢжңүдёҖдёӘ CMakeLists, еңЁйЎ№зӣ®зҡ„ж №зӣ®еҪ•ж–°е»әдёҖдёӘеҸ« build зҡ„ж–Үд»¶еӨ№, з„¶еҗҺиҝӣе…Ҙ build ж–Үд»¶еӨ№еҶ…, жү§иЎҢе‘Ҫд»Ө cmake .. еҚіеҸҜ.

CMakeListsе®һдҫӢи®Іи§Ј

жҜ”еҰӮжңүдёҖдёӘзӣ®еҪ•з»“жһ„еҰӮдёӢзҡ„йЎ№зӣ® :

    в”Ӯ  в”ңв”Җa.h
    в”Ӯ  в”ңв”Җa.cpp
    в”Ӯ  в”ңв”Җb.h
    в”Ӯ  в”ңв”Җb.cpp
    в”Ӯ  в”ңв”ҖTestFolder
    в”Ӯ  в”Ӯ  в”ңв”Җa.h
    в”Ӯ  в”Ӯ  в”ңв”Җa.cpp
    в”Ӯ  в”Ӯ  в”ңв”Җb.h
    в”Ӯ  в”Ӯ  в”ңв”Җb.cpp
    в”Ӯ  в””в”Җ...
    в”ңв”ҖTool

иҝҷжҳҜдёҖдёӘжҜ”иҫғйҖҡз”Ёзҡ„CMakeLists.txt :

еҜ№дәҺеғҸдёҠйқўиҝҷж ·дёҖдёӘCMakeзҡ„CMakeListsжқҘиҜҙ, йңҖиҰҒзқҖйҮҚи§ЈйҮҠзҡ„жңүд»ҘдёӢеҮ зӮ№ :

  • add_definitions(-std=c++11) иҝҷеҸҘжҳҜдёәдәҶи§ЈеҶі linux й»ҳи®ӨдёҚж”ҜжҢҒ c++11 зҡ„й—®йўҳ

  • set_target_properties(${PROJ_NAME} PROPERTIES LINK_FLAGS "/SUBSYSTEM:WINDOWS") # works for all build modes иҝҷеҸҘжҳҜдёәдәҶи§ЈеҶіWinMainзҡ„й—®йўҳ, еҗҰеҲҷеңЁvsе№іеҸ°дјҡжҠҘmainйқһжі•еј•з”Ёзҡ„й—®йўҳ. иҝҷеҸҘдјҡеҪұе“ҚеҲ°vsзҡ„ вҖңйЎ№зӣ®еұһжҖ§-й“ҫжҺҘеҷЁ-зі»з»ҹ-еӯҗзі»з»ҹвҖқ

  • target_link_libraries(${PROJ_NAME} wsock32 ws2_32) иҝҷеҸҘжҳҜдёәдәҶеңЁvsдёӢй“ҫжҺҘws2_32еә“, windowsйңҖиҰҒй“ҫжҺҘиҝҷдёӘеә“жүҚиғҪз”Ёsocket. иҝҷеҸҘдјҡеҪұе“ҚеҲ°vsзҡ„ вҖңйЎ№зӣ®еұһжҖ§-й“ҫжҺҘеҷЁ-иҫ“е…Ҙ-йҷ„еҠ дҫқиө–йЎ№вҖқ

  • file(GLOB_RECURSE CURRENT_HEADERS *.h *.hpp) source_group("Include" FILES ${CURRENT_HEADERS}) иҝҷеҸҘжҳҜдёәдәҶи§ЈеҶіеңЁvsдёӢдёҚжҳҫзӨәеӨҙж–Үд»¶зҡ„й—®йўҳ

  • set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -w")
    -wзҡ„ж„ҸжҖқжҳҜе…ій—ӯзј–иҜ‘ж—¶зҡ„иӯҰе‘ҠпјҢд№ҹе°ұжҳҜзј–иҜ‘еҗҺдёҚжҳҫзӨәд»»дҪ•warningпјҢеӣ дёәжңүж—¶еңЁзј–иҜ‘д№ӢеҗҺзј–иҜ‘еҷЁдјҡжҳҫзӨәдёҖдәӣдҫӢеҰӮж•°жҚ®иҪ¬жҚўд№Ӣзұ»зҡ„иӯҰе‘ҠпјҢиҝҷдәӣиӯҰе‘ҠжҳҜжҲ‘д»¬е№іж—¶еҸҜд»ҘеҝҪз•Ҙзҡ„гҖӮ
    -WallйҖүйЎ№ж„ҸжҖқжҳҜзј–иҜ‘еҗҺжҳҫзӨәжүҖжңүиӯҰе‘ҠгҖӮ
    -WйҖүйЎ№зұ»дјј-WallпјҢдјҡжҳҫзӨәиӯҰе‘ҠпјҢдҪҶжҳҜеҸӘжҳҫзӨәзј–иҜ‘еҷЁи®ӨдёәдјҡеҮәзҺ°й”ҷиҜҜзҡ„иӯҰе‘ҠгҖӮ
    еңЁзј–иҜ‘дёҖдәӣйЎ№зӣ®зҡ„ж—¶еҖҷеҸҜд»Ҙ-Wе’Ң-WallйҖүйЎ№дёҖиө·дҪҝз”ЁгҖӮиҝҷйҮҢеҸҜд»ҘжҹҘзңӢgccзҡ„еҗ„з§ҚиӯҰе‘Ҡзә§еҲ«.
